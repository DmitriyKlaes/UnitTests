## Объяснение того, какие сценарии покрыты тестами и почему я выбрал именно эти сценарии

##### Класс ListFiller:
```
fillListRandomNumbersSizeTest() {
    Юнит-тест
    Этот тест проверяет, что итоговый список имеет 
    заявленную длину.
    Сценарий я выбрал как проверку функциональности метода.
}
```
```
fillListRandomNumbersValidateContentTest() {
    Юнит-тест
    Этот тест проверяет, что все значения в списке 
    находятся в пределах заявленных границ.
    Сценарий я выбрал как проверку функциональности метода
}
```
```
fillListRandomNumbersNullArgumentTest() {
    Юнит-тест
    Этот тест проверяет выбрасывается ли исключение, 
    если в качестве аргумента передан объект без ссылки.
    Так же он сверяет сообщение исключения с ожидаемым.
    Сценарий я выбрал как проверку негативного исхода
    работы метода.    
}
```
